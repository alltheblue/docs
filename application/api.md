## 支持功能

1. 支持接口断言类型检测
2. 支持存储变量
3. 支持导出 interface
4. 支持导出测试报告
5. 支持自定义返回成功值修改
6. 支持自定义函数
7. 支持分组选择
8. 支持接口、分组拖拽排序
9. 支持全局配置请求头，分组配置请求头请求参数
10. 支持分组，单个接口复制
11. 支持关联性接口测试
12. 支持导入导出 API 文件
